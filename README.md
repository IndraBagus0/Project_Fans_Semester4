## Installation
1. Clone Repo
    ```console
    git clone https://github.com/IndraBagus0/Project_Fans_Semester4.git
    ```

2. Run

   ```console
   composer install
   ```
   ```console
   php artisan key:generate
   ```
   ```console
   php artisan migrate
   or
   php artisan migrate:refresh
   ```
   ```console
   php artisan db:seed
   ```

3. Run server
   ```console
   php artisan serve
   ```
